### Install SQANTI
could not get sqanti3 installed on secretariat so transferred pigeon and ref docs to palmetto and continued there
```
wget https://github.com/ConesaLab/SQANTI3/releases/download/v5.5.4/SQANTI3_v5.5.4.zip
mkdir sqanti3
unzip SQANTI3_v5.5.4.zip -d sqanti3
cd sqanti3
conda env create -f SQANTI3.conda_env.yml
y
conda activate sqanti3
```
